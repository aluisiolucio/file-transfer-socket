# file-transfer-socket

## Pré-requisitos

- Python 3.11 ou superior
- Pip para instalar as dependências

### Instalando dependências

```
pip install -r requirements.txt
```
ou
```
pip3 install -r requirements.txt
```

### Executando servidor

```
python server.py
```
ou
```
python3 server.py
```

### Executando cliente (UI)

```
python app.py
```
ou
```
python3 app.py
```
